
# Chat Feed

A small chat app I made in a few hours as practice.


## Note

Avatars are taken from Gravatar profile linked to the inserted mail.


## Technologies

- MongoDB
- Express
- React
- Node.js
- Socket.io

You can check it out here: https://chatfeed.herokuapp.com/.
